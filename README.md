# 象棋作弊系统

基于最强开源象棋引擎 Pikafish 的可视化辅助分析系统。AI 帮你计算我方走法，你手动输入对方走法，全程点击棋子操作。

## 快速启动

```bash
# 双击启动
启动可视化版.bat

# 或命令行
python main_visual.py
```

无需安装第三方依赖，使用包含 tkinter 的标准 Python 即可运行。

## 使用流程

1. **选择先手** —— 选「我方先手」或「对方先手」
2. **开始对局** —— 点击「开始对局」，引擎启动并开始辅助
3. **我方回合** —— AI 自动算出我方最佳走法并落子，在「AI 建议」区告诉你该走哪步
4. **对方回合** —— 点击棋盘输入对方实际走的棋（先点棋子，再点目标位置）
5. 重复 3～4

## 功能按钮

- **开始对局** —— 启动引擎并按所选先手开始
- **重算我方走法** —— 让 AI 重新计算当前我方走法
- **手动输入我方** —— 撤销 AI 的走法，改由自己点击输入我方走法
- **对调棋盘** —— 上下翻转棋盘视图
- **回退一步** —— 撤销最后一步
- **重置对局** —— 清空棋局，回到未开始状态

## 走法规则校验

落子前会校验是否符合行棋规则（车走直线、马走日蹩马腿、炮隔子吃、相不过河、仕/帅不出九宫、兵过河才能平移、将帅不照面等），非法走法会被拦截并提示原因。

## 项目结构

```
xiangqi_game/
├── main_visual.py        # 主程序（可视化 GUI）
├── engine_controller.py  # 引擎 UCI 通信
├── board.py              # 走法记录
├── requirements.txt      # Python 依赖
├── 启动可视化版.bat       # 一键启动脚本
└── engine/
    ├── pikafish.exe      # Pikafish 引擎
    └── pikafish.nnue     # NNUE 神经网络权重
```

## 技术信息

- **引擎**: Pikafish（基于 Stockfish 的开源象棋引擎，NNUE 评估）
- **GUI**: Tkinter（Python 内置）
- **通信协议**: UCI
- **坐标系统**: ICCS（如 `h2e2`）

引擎参数在 `main_visual.py` 中调整：搜索深度 `depth`、思考时间 `time_ms`。
